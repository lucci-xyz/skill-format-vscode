# Changelog

## 0.1.0

Initial release.

- YAML frontmatter formatting with field ordering and block scalar descriptions
- Markdown body formatting (section spacing, lists, fences, whitespace)
- Inline diagnostics for common skill authoring mistakes
- Format on save
- Command palette: `Skill Format: Format SKILL.md`
- Status bar indicator for active SKILL.md files
- Snippets: `skill:init`, `skill:section`, `skill:sub`, `skill:ref`
- `skill-md` language registration for SKILL.md files
